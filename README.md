# Riverview Caledonia Phase 2 — Preview

Marketing site preview for Riverview Caledonia Phase 2 (Elite Developments).

## View live

Live preview: https://YOUR-USERNAME.github.io/riverview-preview/

(Replace `YOUR-USERNAME` with your GitHub username once Pages is configured.)

## Files

- `index.html` — the site
- `hero_video.mp4` + `hero_poster.jpg` — looping hero background video
- `construction_video.mp4` + `construction_poster.jpg` — click-to-play in closing section

All asset files must sit in the same directory as `index.html` — relative paths.

## Deploy to GitHub Pages

1. Push this folder to a GitHub repo
2. Repo → Settings → Pages
3. Source: "Deploy from a branch"
4. Branch: `main` (or `master`) → `/ (root)` → Save
5. Wait 1-2 minutes
6. Visit the URL shown in the Pages settings

## Privacy

The HTML includes a `<meta name="robots" content="noindex, nofollow">` tag so the preview won't appear in Google search results. The page is still accessible to anyone who has the URL.
